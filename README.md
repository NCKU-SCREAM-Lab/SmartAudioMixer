# SmartAudioMixer 主畫面開發

### 功能
---
* File
  * 讀檔
    * Import Project
  * 寫檔
    * Save As
      * also, Save As Wav
    * Save Project
  * 新增檔案
    * New Project
* Track
  * For track
    * New / Delete  Track
    * New / Delete  Audio Clip
    * Undo / Redo
    * TimeBar
    * ScrollX / Scale
    * Solo / Mute
    * VST3
  * For clip
    * Show Waveform
    * Choose / Move / Play & Stop
  * Plugin
